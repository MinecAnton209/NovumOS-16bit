## Введение

NovumOS-16bit работает на пользовательском 16-битном процессоре. Все периферийные устройства подключаются через простую шину I/O, используя инструкции `IN` и `OUT`.

---

## Карта I/O портов

| Порт | Устройство | Направление | Описание |
|------|-----------|-------------|----------|
| `0x00` | UART | R/W | Терминал (IN = чтение символа, OUT = запись символа) |
| `0x01` | Таймер | Read | Счётчик тиков (младшие 16 бит) |
| `0x02` | Клавиатура | Read | Скан-код (0 если буфер пуст) |

---

## Инструкции IN и OUT

### Инструкция IN (чтение из порта)

Инструкция `IN` читает 16-битное значение из порта ввода-вывода в регистр.

**Синтаксис:** `IN AX, порт`

**Примеры:**
```asm
IN AX, 0x00    ; Чтение символа из UART (0 = нет данных)
IN AX, 0x01    ; Чтение счётчика тиков таймера
IN AX, 0x02    ; Чтение скан-кода клавиатуры (0 = пусто)
```

### Инструкция OUT (запись в порт)

Инструкция `OUT` записывает 16-битное значение из регистра в порт ввода-вывода.

**Синтаксис:** `OUT порт, AX`

**Примеры:**
```asm
OUT 0x00, AX   ; Запись символа в терминал UART
```

---

## UART (Порт 0x00)

Простой порт терминального ввода-вывода. Без регистров статуса, без настройки скорости передачи.

- **OUT 0x00, символ** — Отправляет символ в буфер вывода терминала
- **IN AX, 0x00** — Читает символ из буфера ввода терминала (возвращает 0 если пусто)

---

## Таймер (Порт 0x01)

Простой счётчик тактов, увеличивающийся на каждом шаге CPU.

- **IN AX, 0x01** — Возвращает младшие 16 бит счётчика тактов

---

## Клавиатура (Порт 0x02)

Буфер скан-кодов клавиатуры.

- **IN AX, 0x02** — Читает следующий скан-код из буфера (возвращает 0 если пусто)
