# Ассемблерные обертки

Высокоуровневые обертки на Zig для ISA NovumOS-16bit. Предоставляют удобные функции для генерации машинного кода без вызова raw encode функций напрямую.

**Источник:** `src/wrappers/asm.zig` (тесты: `src/wrappers/test.zig`)

---

## Использование

```zig
const asm_ = @import("wrappers/asm.zig");

const program = [_]u16{
    asm_.nop(),
    @truncate(asm_.mov_imm(.AX, 0x1234)),
    @truncate(asm_.mov_imm(.BX, 0x5678)),
    asm_.add(.AX, .BX),
    asm_.push(.AX),
    asm_.call(0x0100),
    asm_.hlt(),
};
```

**Примечание:** `mov_imm` возвращает `u32` (32-битная инструкция). Используйте `@truncate` при вставке в массив `u16` — сохраняются только младшие 16 бит.

---

## Инструкции памяти

| Функция | Возвращает | Описание |
|---------|-----------|----------|
| `mov_imm(dst, imm)` | `u32` | `MOV dst, imm16` — загрузка 16-битного числа |
| `mov(dst, src)` | `u16` | `MOV dst, src` — между регистрами |
| `mov_load(dst, src)` | `u16` | `MOV dst, [src]` — чтение из памяти |
| `mov_load_off(dst, src, off)` | `u32` | `MOV dst, [src+off]` — чтение со смещением **(FIXME: кодировка сломана)** |

## ALU инструкции

Все ALU функции возвращают `u16`.

| Функция | Инструкция | Описание |
|---------|------------|----------|
| `add(dst, src)` | `ADD dst, src` | dst = dst + src |
| `sub(dst, src)` | `SUB dst, src` | dst = dst - src |
| `cmp(dst, src)` | `CMP dst, src` | Сравнение (только флаги) |
| `and(dst, src)` | `AND dst, src` | dst = dst AND src |
| `or(dst, src)` | `OR dst, src` | dst = dst OR src |
| `xor(dst, src)` | `XOR dst, src` | dst = dst XOR src |
| `shl(dst, src)` | `SHL dst, src` | dst = dst << src |
| `shr(dst, src)` | `SHR dst, src` | dst = dst >> src |
| `inc(dst)` | `INC dst` | dst = dst + 1 |
| `dec(dst)` | `DEC dst` | dst = dst - 1 |
| `not(dst)` | `NOT dst` | dst = NOT dst |
| `neg(dst)` | `NEG dst` | dst = 0 - dst |
| `xchg(dst, src)` | `XCHG dst, src` | Обмен dst и src |
| `adc(dst, src)` | `ADC dst, src` | dst = dst + src + Carry |
| `sbb(dst, src)` | `SBB dst, src` | dst = dst - src - Carry |
| `test_(dst, src)` | `TEST dst, src` | Побитовое AND (только флаги) |

**Примечание:** `test_` имеет суффикс `_` потому что `test` — ключевое слово Zig.

## Сдвиг на константу

Использует CX для хранения количества сдвигов.

| Функция | Возвращает | Описание |
|---------|-----------|----------|
| `shl_imm(reg, imm)` | `[2]u32` | `MOV CX, imm; SHL reg, CX` |
| `shr_imm(reg, imm)` | `[2]u32` | `MOV CX, imm; SHR reg, CX` |

---

## Алиасы Compare + Branch

Удобные алиасы для условных переходов. Все возвращают `u32`.

| Функция | Алиас для | Условие |
|---------|-----------|---------|
| `je(target)` | `JZ` | Равно (флаг Zero установлен) |
| `jne(target)` | `JNZ` | Не равно (флаг Zero сброшен) |
| `jge(target)` | `JNS` | Больше или равно (знаковый) |
| `jl(target)` | `JS` | Меньше (знаковый) |
| `ja(target)` | `JNC` | Выше (беззнаковый) |
| `jbe(target)` | `JC` | Ниже или равно (беззнаковый) |

Оригинальные переходы тоже доступны: `jz`, `jnz`, `jc`, `jnc`, `js`, `jns`.

## Инструкции стека

| Функция | Возвращает | Описание |
|---------|-----------|----------|
| `push(reg)` | `u16` | PUSH одного регистра |
| `pop(reg)` | `u16` | POP одного регистра |
| `push2(a, b)` | `[2]u16` | PUSH двух регистров |
| `pop2(a, b)` | `[2]u16` | POP двух регистров (обратный порядок) |
| `push4(a, b, c, d)` | `[4]u16` | PUSH четырёх регистров |
| `pop4(a, b, c, d)` | `[4]u16` | POP четырёх регистров (обратный порядок) |

**Примечание:** `pop2` и `pop4` извлекают в обратном порядке относительно push (LIFO).

---

## Управление потоком

| Функция | Возвращает | Описание |
|---------|-----------|----------|
| `jmp(target)` | `u32` | Безусловный переход |
| `jmp_reg(reg)` | `u16` | Переход по адресу в регистре |
| `call(target)` | `u32` | Вызов подпрограммы |
| `ret()` | `u16` | Возврат из подпрограммы |

---

## I/O инструкции

| Функция | Возвращает | Описание |
|---------|-----------|----------|
| `out(port, reg)` | `u32` | Запись регистра в I/O порт |
| `in(reg, port)` | `u32` | Чтение I/O порта в регистр |
| `out_char(reg)` | `u32` | Вывод символа в UART (порт 0x00) |
| `in_char(reg)` | `u32` | Ввод символа из UART (порт 0x00) |

---

## Системные инструкции

| Функция | Возвращает | Описание |
|---------|-----------|----------|
| `nop()` | `u16` | Без операции |
| `hlt()` | `u16` | Остановка CPU |
| `int(vector)` | `u32` | Программное прерывание (обработчик = vector × 4) |
| `iret()` | `u16` | Возврат из прерывания |

---

## Утилиты

| Функция | Возвращает | Описание |
|---------|-----------|----------|
| `clear(reg)` | `u16` | Очистка регистра (XOR reg, reg) |
| `nops(n)` | `[n]u16` | Вставить n инструкций NOP |
